BlueClick - Auto Clicker Application
=====================================

This is a standalone executable version of the BlueClick auto clicker application.

## What's Included:
- BlueClick.exe - The main application
- assets/ - Application assets (icons)
- coords/ - Coordinate files for click scripts

## How to Use:
1. Run BlueClick.exe to start the application
2. Use the interface to create and manage click scripts
3. Save your coordinate scripts in the coords folder
4. Use hotkeys to control the auto clicker

## Features:
- Record mouse coordinates and keyboard keys
- Set delays between actions
- Create repeatable click sequences
- Hotkey controls
- Dark/light theme support

## Note:
- The application will create a config.json file on first run to store your settings
- Coordinate files are stored in the coords folder
- The app can be minimized to system tray
