# BlueClickByS - Modern Auto Clicker

BlueClickByS adalah aplikasi autoclicker modern yang dirancang untuk mengotomatiskan klik mouse dan input keyboard. Aplikasi ini menyediakan berbagai fitur canggih untuk membuat dan mengelola skrip otomatisasi dengan mudah.

## Fitur Utama

- **Autoclicker** - Klik otomatis pada koordinat tertentu dengan jeda yang dapat disesuaikan
- **Input Keyboard Otomatis** - Kirim input keyboard otomatis sesuai skrip
- **Pengaturan Jeda** - Atur jeda antar aksi dalam milidetik
- **Pengulangan** - Atur jumlah pengulangan otomatis untuk skrip
- **Penyimpanan Skrip** - Simpan dan muat skrip dalam format JSON
- **Pengaturan Tema** - Mode gelap dan terang
- **Hotkey Global** - Toggle otomatis dengan hotkey khusus
- **Fungsi Anti-Deteksi** - Menambahkan variasi acak ke koordinat untuk menghindari deteksi

## Instalasi

### Dari Executable

- Unduh Autoclick.zip di https://drive.google.com/file/d/1DL0KOMlgu3UBMOukDgdVeM5_9U7Z70Uk/view?usp=sharing
- Unzip Autoclick.zip
- jalankan `BlueClick.exe`

## Penggunaan

### Dasar

1. Buka aplikasi
2. Gunakan tombol **"Pick"** untuk memilih koordinat dari layar
3. Atur nilai X dan Y secara manual jika diperlukan
4. Atur **Delay (ms)** dan **Repetisi**
5. Klik **"Tambah"** untuk menambahkan aksi ke daftar
6. Gunakan tombol **"Mulai"** untuk memulai autoclick

### Membuat Skrip

1. Tambahkan beberapa aksi klik ke daftar
2. Tambahkan input keyboard jika diperlukan
3. Gunakan tombol **"Simpan Script"** untuk menyimpan skrip
4. Skrip akan disimpan di folder `coords/`

### Memuat Skrip

1. Pilih skrip dari daftar **"Daftar Script"**
2. Klik untuk memuat skrip yang dipilih
3. Konfirmasi jika diminta

### Hotkey

- Gunakan hotkey toggle (default: `ctrl+s`) untuk mulai/berhenti
- Hotkey toggle bisa di atur di pengaturan atau menekan tombol set

## Fitur Lanjutan

### Anti-Deteksi

Aplikasi ini dilengkapi fitur anti-detection yang menambahkan variasi acak (±1-5 pixel) ke setiap koordinat ketika skrip dijalankan untuk membuat klik terlihat lebih alami.

### Edit Delay

- Klik kanan pada aksi untuk membuka menu konteks
- Pilih **"Edit Delay"** untuk mengubah jeda aksi
- Atau klik dua kali pada aksi untuk langsung mengedit delay

### Pengaturan

- Buka menu **"Pengaturan"** → **"Pengaturan Tema & Hotkey"**
- Ubah tema antara gelap dan terang
- Atur hotkey toggle sesuai kebutuhan

## Peringatan Keamanan

⚠️ **Penting**: Aplikasi ini mungkin ditandai sebagai malware oleh Windows Defender atau antivirus lainnya karena fungsionalitasnya yang mengotomatiskan input pengguna (mouse dan keyboard). Ini adalah respons keamanan standar karena alat seperti ini bisa disalahgunakan. Pastikan Anda:

- Hanya mengunduh dari sumber terpercaya
- Menonaktifkan sementara antivirus saat pertama kali menjalankan, atau
- Menambahkan pengecualian untuk aplikasi ini di pengaturan antivirus

Aplikasi ini dirancang untuk penggunaan yang sah dan tidak mengandung malware sebenarnya.

## Troubleshooting

### Aplikasi tidak bisa dibuka

- Periksa pengaturan antivirus

### Hotkey tidak berfungsi

- Pastikan tidak ada aplikasi lain yang menggunakan hotkey yang sama
- Coba hotkey lain di pengaturan

### Aplikasi tetap berjalan di latar belakang

- Gunakan tombol 'X' untuk menutup aplikasi sepenuhnya
- Atau hentikan dari Task Manager jika perlu
